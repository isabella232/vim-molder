# vim-molder

Minimal File Explorer

## Usage

```
$ vim /path/to/directory/
```

## Installation

For [vim-plug](https://github.com/junegunn/vim-plug) plugin manager:

```
Plug 'mattn/vim-molder'
```

## Options

* Show dotfiles and dotdirs:

    ````vim
    let g:molder_show_hidden = 1
    ````
